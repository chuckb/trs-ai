# Agent instructions (TRS-AI)

## Host Python

Before running `pip`, `pytest`, or editable installs for **MBASIC** in this repo, activate the workstation venv (see [README.md](README.md) “Host Python”):

```bash
source ~/pyenvs/trs-ai/bin/activate
cd mbasic
```

Do not install packages into the system Python for this project’s development workflow.
